Международные провайдеры

Международные провайдеры доступны по всему миру, имеют локализацию для России и выбор валюты оплаты.

Microsoft Azure

  • Официальный сайт

Microsoft Azure - один из самых популярных провайдеров облачных решений в мире. Azure используют для реализации своих задач мировые и российские компании, такие как:

  • ПАО "ВымпелКом" (Beeline)
  • Финансовый университет при Правительстве Российской Федерации
  • Nordgold
  • ОАО «Международный аэропорт Шереметьево»
  • Уральский федеральный университет

Microsoft предоставляет возможность создания множества продуктов на облачной базе, самые популярные из которых:

  • Виртуальные машины Windows Server и Linux
  • Виртульная сеть
  • Шлюз VPN
  • AppService - создание приложений для любой платформы онлайн
  • Базы данных SQL

Также Microsoft предоставляет огромное количество обучающих видео по Azure, большинство из которых имеют русские субтитры.

Цены на услуги

Цены на услуги Microsoft Azure варьируются в зависимости от продукта и расположения сервера. Например, самый дешевый стандартный тариф (1,06 рубль/час) на виртуальную машину Linux/Windows при выборе сервера, находящегося на западе Японии. Такое расположение сервера вполне подходит, если Вы находитесь в Сибири или на Дальнем Востоке, так как задержка доступа не будет сильно отличаться от сервера в Европе.
